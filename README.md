## APIs

### CSG Forte REST API
CSG Forte provides full-stack REST APIs for payment processing within a PCI-compliant architecture. The API enables merchants and partners to create and update credit card, echeck, and scheduled transactions, securely manage customer and payment data, and query settlement information. Authentication uses standard HTTP credential headers.

**Human URL:** [https://developers.forte.net/](https://developers.forte.net/)

### CSG Forte.js
Forte.js is a JavaScript library for secure browser-based payment tokenization. It enables web applications to collect and tokenize payment card data client-side before submitting to Forte's payment API, reducing PCI scope.

**Human URL:** [https://developers.forte.net/forte-js/](https://developers.forte.net/forte-js/)
